Directivas

Las directivas son atributos especiales con el prefijo v-

A las directivas les podemos pasar directivas

Las directivas van dentro de las etiquetas de html

Directiva Ejemplo ¿Qué hace?

v-bind

 < a v-bind:href="miLink">Este es el link< /a> Recibe un argumento la propiedad que se bindeará
y a esa propiedad se le asigna una variable para que
sea de manera dinámica.

v-cloak

v-text

v-html

v-model

v-if

 < p v-if = "miVar"> mi texto< p/>
< p v-if = "miVar == 5"> mi texto< p/>
< p v-if = "x==y"> mi texto< p/>

Si se cumple la condicion de la variable
se muestra el parrafo o no se muestra.

v-else

v-else-if

v-for

v-once

v-on

 < input v-on:keyup="enviar">
< input v-on:keyup.enter="enviar">

v-pre

v-show
